# crna-project

리액트 네이티브에서 detox 세팅해보기

# CRNA

```
$ npm i -g create-react-native-app
$ create-react-native-app crna-project
$ cd crna-project
$ npm install

```

# detox setting

```
$ xcode-select --install
$ brew tap wix/brew
$ brew install applesimutils
$ npm install -g detox-cli

$ yarn add --dev detox jest-circus
$ detox init -r jest
```

# prettier / lint

```
npm run lint
npx prettier --write .
```
